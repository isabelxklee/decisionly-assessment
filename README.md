# Decisionly Assessment

## Getting Started

1. Install Node.js.

2. Git clone this repository.

3. Install the project packages.

```
$ npm install
```

4. Start development server.

```
$ npm run dev
```

5. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Tech Stack

- Next.js
- TypeScript
- Open AI
- PDF2JSON, FilePond
